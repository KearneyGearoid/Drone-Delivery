Drone Delivery Service

Use Case 14: Drone Status

Drone status updates the system about the status of a drone

Drone status is used to locate and update the status of a drone. The Drone Fleet Manager will run the function and search for a drone, the program will search through the system to find the drone's information and request an up date on the drones location, stage of a delivery and what charge it has left from the drone

Track Drone

Primary Actor: Drone Fleet Manager

Goal in Context: A Drone Fleet Manager to veiw comprehesive data on drone locations and status.

Scope: Business - It is required as part of a drone delivery system

Level: User goal

Stakeholders and Interests

Drone Fleet Manager: wants to see where the drone(s) currently in an easy way.

Pre Conditions:The user is a Drone Fleet Manager

Post Conditions: None

Success Scenario:

  1. Manager: runs the 'Drone Status' function.
  2. System: Displays displays the interface
  3. Manager: enteres in a drones identification
  4. System: searches for a drone in the system
    5. Manager: confirms that it is the drone they are looking for
  5. System: sends out a request to the drone to update its status
  6. Drone: replies with status on location, charge and stage of delivery
  7. System: retirieves the information and displays it
  8. Manager: closes function

    9. Extentions:

  9. 3a: there are no drone registered on the system with that identifier.
  10. 8a: the system does not retrieve a response
