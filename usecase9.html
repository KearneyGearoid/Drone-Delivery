Use Case 9: Add Aerial Charging Point

An aerial charging point can be registered and added to the system. The coordinates of the aerial charging points are added by the system admin.

When an aerial charging point is erected in a location, that aeral charging point is assigned a unigue identifier. This unigue identifier along with the location, and type of charger is registered in the system by the system admin using the aerial charging point method. This will take in the information and register it into the database.
